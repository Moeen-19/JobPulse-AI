# Job Market Intelligence Platform - Website

A modern, professional website for the Job Market Intelligence Platform with a sleek dark theme and grid background aesthetic.

## Features

- **Modern Dark Theme**: Professional black background with grid overlay
- **Smooth Animations**: Scroll-based animations and hover effects
- **Responsive Design**: Works on all device sizes
- **Interactive Elements**: Dynamic hover states and transitions
- **Clean Typography**: Using Inter font family for modern look

## Pages

1. **Home** (`index.html`) - Main landing page with:
   - Hero section with key statistics
   - Skills discovery section
   - Live market insights
   - User testimonials
   - Feedback section
   - Footer with navigation

2. **Job Discovery** (`job-discovery.html`) - Coming soon
3. **Market Analysis** (`market-analysis.html`) - Coming soon
4. **About Us** (`about.html`) - Coming soon

## How to Run

Simply open `index.html` in your web browser. No build process required.

## Color Scheme

- Background: `#0a0a0a` (Dark black)
- Cards: `#1a1a1a` (Slightly lighter black)
- Primary Accent: `#00ff88` (Neon green)
- Secondary Accent: `#00d4ff` (Cyan blue)
- Text: `#ffffff` (White) and `#a0a0a0` (Gray)
